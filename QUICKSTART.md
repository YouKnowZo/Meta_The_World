# 🚀 Quick Start Guide - Meta World

Get your virtual life started in under 5 minutes!

## Step 1: Install Dependencies

```bash
npm install
```

Wait for all packages to download (takes ~1-2 minutes).

## Step 2: Start the Server

```bash
npm run dev
```

The server will start on `http://localhost:3000`

## Step 3: Open Your Browser

Navigate to: **http://localhost:3000**

## Step 4: Create Your Identity

1. You'll see the welcome screen
2. Enter your name (whatever you want to be called in Meta World)
3. Click "Begin Your Journey"

## Step 5: Start Exploring!

### Your First Actions:

#### 🎮 Movement
- Press **W, A, S, D** to move around
- Hold **Shift** to sprint
- Explore the 3D world!

#### 🏠 Become a Real Estate Agent

1. Click the **"Properties"** button at the bottom
2. Browse the 50+ available properties
3. Click on any property to see details
4. Click **"Facilitate Sale"** to earn your first commission!
5. Watch your MetaCoins grow! 💰

#### 💼 Try Other Careers

1. Click the **"Career"** button at the bottom
2. Explore 9 different career paths
3. Switch to any career you like
4. Each career has unique opportunities!

#### 💬 Connect with Others (Future Feature)

- Click the chat icon in the bottom right
- Send messages to the global chat
- Build your network!

## What You Get

- **50,000 MetaCoins** to start
- **Level 1 Real Estate Agent** role
- **3% commission rate** on all sales
- **Access to 50+ properties** from studios to mansions
- **Dynamic 3D world** with day/night cycles

## Tips for Your First Day

1. **Don't buy immediately** - As an agent, you earn commissions without spending!
2. **Facilitate 5 sales** - Build up your wallet first
3. **Level up** - Each sale gives you XP
4. **Then buy property** - Use your earnings to invest
5. **Diversify** - Own different types of properties

## Quick Reference

### Controls
- `W` - Forward
- `A` - Left
- `S` - Back
- `D` - Right
- `Shift` - Sprint
- `Mouse` - Look around (click buildings to select)

### UI Buttons
- **Properties** - Real estate marketplace
- **Career** - Change professions and view stats
- **Chat** - Social features (bottom right)

### Your Stats (Right Side)
- Properties Owned
- Commission Rate
- Reputation
- Experience Points

### Top Bar
- Your name and level
- MetaCoin wallet balance
- Time of day
- Weather conditions

## Troubleshooting

### Port already in use?
```bash
# Try a different port
npm run dev -- --port 3001
```

### Dependencies not installing?
```bash
# Clear cache and try again
npm cache clean --force
npm install
```

### Performance issues?
- Close other browser tabs
- Use Chrome or Firefox for best performance
- Reduce window size if needed
