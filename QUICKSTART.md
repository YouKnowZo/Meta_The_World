# Quick Start Guide

## Installation & Setup

1. **Install all dependencies:**
```bash
npm run install-all
```

2. **Configure environment:**
```bash
cp .env.example .env
# Edit .env with your MongoDB connection string
```

3. **Start MongoDB** (if using local):
```bash
mongod
```

4. **Run the application:**
```bash
npm run dev
```

This starts:
- Backend server on `http://localhost:5000`
- Frontend app on `http://localhost:3000`

## First Steps

1. **Register an account** at `http://localhost:3000/register`
2. **Enter the world** - You'll start with 10,000 MTC
3. **Explore** using WASD keys
4. **Become an agent** - Visit the menu and click "Become an Agent"
5. **Start earning** - Help facilitate property sales and earn commissions!

## Key Features to Try

- **Explore Properties**: Click on red property markers in the world
- **Purchase Properties**: Buy properties you like
- **List Your Properties**: As an owner, list properties for sale
- **Earn Commissions**: As an agent, earn 5% on every sale you facilitate
- **Track Performance**: View your stats in the Dashboard and Agent Dashboard

Enjoy your virtual world! 🌍✨
