# Contributing to Meta The World

Thank you for your interest in contributing to Meta The World! As we are currently in the **Genesis** phase, we are focused on establishing the core infrastructure.

## How to Contribute

1.  **Report Bugs**: Use GitHub Issues to report bugs. Please provide a detailed description and steps to reproduce.
2.  **Suggest Features**: We welcome feature requests! Open an issue and use the "Feature Request" label.
3.  **Submit Pull Requests**:
    - Fork the repository.
    - Create a new branch for your changes.
    - Ensure your code follows the project's style and conventions.
    - Submit a pull request to the `main` branch.

## Development Guidelines

- **Code Quality**: Ensure all code is well-tested and documented.
- **Commit Messages**: Use descriptive commit messages.
- **Testing**: Run all tests before submitting a pull request.
