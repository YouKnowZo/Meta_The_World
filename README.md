# Meta The World

A hyper-realistic virtual world where you can be anything you want to be - a place better than reality where opportunities are limitless.

## Features

### 🏠 Virtual Real Estate System
- **Property Listings**: Browse and explore virtual properties in a 3D world
- **Property Ownership**: Purchase and own virtual real estate
- **Hyper-Realistic Graphics**: Advanced 3D rendering with shadows, lighting, and realistic terrain
- **Property Types**: Residential, Commercial, Mansions, Penthouses, and Land

### 🏢 Real Estate Agent Career
- **Become an Agent**: Transform into a virtual real estate agent
- **5% Commission**: Earn 5% commission on every transaction you facilitate
- **Transaction Tracking**: Monitor all your sales and earnings
- **Agent Dashboard**: Comprehensive dashboard showing your performance
- **Property Management**: List properties and help buyers find their perfect virtual home

### 🌍 Virtual World Features
- **3D Environment**: Explore a vast 3D world with realistic terrain
- **Dynamic Buildings**: Diverse building types scattered across the world
- **Other Users**: See other avatars exploring the world
- **Real-time Updates**: Live updates via WebSocket connections
- **Interactive UI**: Beautiful, modern interface for all interactions

### 🎮 Activities & Opportunities
- **Social Hub**: Connect with other users
- **Events**: Participate in virtual events
- **Virtual Jobs**: Explore various career opportunities beyond real estate
- **Endless Possibilities**: Be what you want to be in this virtual world

## Technology Stack

- **Frontend**: React, Three.js, React Three Fiber
- **Backend**: Node.js, Express, Socket.io
- **3D Graphics**: Three.js with advanced rendering
- **State Management**: Zustand
- **Real-time Communication**: WebSocket (Socket.io)

## Getting Started

### Prerequisites
- Node.js 16+ and npm

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

This will start both the frontend (Vite) on port 3000 and the backend server on port 3001.

3. Open your browser and navigate to:
```
http://localhost:3000
```

### Production Build

```bash
npm run build
npm start
```

## How to Use

1. **Explore the World**: Use mouse/trackpad to navigate the 3D world
2. **Become an Agent**: Click "Become an Agent" to start your real estate career
3. **Browse Properties**: Click on property markers (green flags) to view details
4. **Purchase Properties**: Buy properties directly from the property panel
5. **List Properties**: Use the menu to list new properties for sale
6. **Track Earnings**: View your agent dashboard to see your commissions and transactions
