## 🎯 Game Mechanics

### Commission Calculation
```
Commission = Property Price × Commission Rate
Default Rate = 3% (0.03)

Examples:
$50,000 × 0.03 = $1,500
$100,000 × 0.03 = $3,000
$500,000 × 0.03 = $15,000
```

### XP Calculation
```
XP Gained = Property Price ÷ 1,000

Examples:
$50,000 sale = 50 XP
$100,000 sale = 100 XP (instant level up!)
$500,000 sale = 500 XP (5 levels!)
```

### Property Pricing
```
Final Price = Base Price × Quality Multiplier
Quality Multiplier = 0.5 + (Quality/100 × 1.5)

Quality 50: Multiplier = 1.25
Quality 75: Multiplier = 1.625
Quality 100: Multiplier = 2.0
```
